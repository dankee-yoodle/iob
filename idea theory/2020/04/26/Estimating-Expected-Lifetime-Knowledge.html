1

Define conceiving of some thing as having it in temporary memory. Define knowing some thing as having it in permanent memory.

Assume that any time some thing is in temporary memory there is some probability it transitions to permanent memory - that any time some thing is conceived there is some probability it is acquired as knowledge.

Define an idea as any thing that can be conceived and acquired as knowledge. Define the set of all ideas as $I$ - all conceivable, knowable things are contained in the set $I$.

Define stimulating a thing as prompting the thing to conceive of an idea.

Define an experience as any thing that can stimulate. Define the set of all experiences as $E$ - all things that can stimulate are contained in the set $E$.

Define a living thing as any thing that is prompted by experiences to conceive of and know of ideas.

Assume that, at every moment a thing is living, it is being stimulated by some experience and conceiving some idea seeded by the experience.

2

Assume that not every living thing is equally likely to acquire an idea as knowledge once it is conceived. Assume that not every idea is equally likely to be acquired as knowledge upon conception by a living thing. Assume these are independent effects.

Assume that a living thing can be assigned a quantity describing how likely it is to acquire a conceived idea as knowledge on average. Assume that every idea can be assigned a quantity describing how likely it is to be acquired as knowledge upon conception by a living thing on average.

Define the quantity assigned to a living thing as the aptitude of the living thing. Define the quantity assigned to an idea as the difficulty of the idea.

3

Assume the probability that a living thing acquires an idea as knowledge upon conceiving it depends only on the aptitude of the living thing and the difficulty of the idea. Explicitly, assume the probability a living thing with aptitude $a$ acquires an idea of difficulty $d$ upon conceiving it is given by

$$p(a,d)=\frac{1}{1+e^{-(a-d)}}$$

Assume that every acquired idea constitutes an amount of knowledge corresponding to the difficulty of the idea. Explicitly, assume that acquiring an idea of difficulty $d$ earns a living thing quanta of knowledge equal to

$$q(d)=e^d$$

A living thing with aptitude $a$ which, stimulated by experiences, conceives of N ideas of difficulties $\vec{d}=d_1,d_2,...,d_N$ over its lifetime will then be expected to acquire an amount of lifetime knowledge $K$ equal to

$$K(a,\vec{d})=\sum_{i=1}^{N}p(a,d_i)q(d_i)=\sum_{i=1}^{N}\frac{e^{d_i}}{1+e^{-(a-d_i)}}$$

These forms of $p(a,d)$ and $q(d)$ were chosen for simplicity, not because they resemble reality particularly well. The proper form might be an emperical question. The conclusions drawn below are particular to the assumptions made here.

4

Assume that the aptitude of a living thing is a random draw from a probability distribution. Explicitly, assume the aptitude of a living thing is a random draw from a normal distribution with a mean and deviation characterizing the subset of the set of all living things the living thing belongs to.

Assume that the difficulty of an idea is a random draw from a probability distribution. Explicitly, assume the difficulty of an idea is a random draw from a normal distribution with a mean and deviation characterizing the subset of the set of all ideas the idea belongs to.

Under these definitions and assumptions, the impact of changing the difficulty of available ideas on the accumulation of knowledge by living things, etc, can be easily approximated.

5

Here I consider a population of 100 living things characterized by a $N(X,1)$ aptitude distribution, where each living thing is assumed to conceive the same set of 100 ideas characterized by a $N(Y,1)$ difficulty distribution for $X$,$Y$ in $-1,0,1$

#collapse-hide

import numpy as np
import pandas as pd
import math
import matplotlib.pyplot as plt
import altair as alt

def prob_acquire(apt,dif):
    return math.exp(apt)/float(math.exp(apt) + math.exp(dif))

def knowledge_acquired(dif):
    return math.exp(dif)


#collapse-hide

num_ideas = 100
num_individuals = 100
idea_dif_std = 1
pop_apt_std = 1


choices1 = [-1,0,1]
choices2 = [-1,0,1]


plot_rows = []
for choice1 in choices1:
    for choice2 in choices2:
    
        pop_apt_mean = choice1
        idea_dif_mean = choice2

        individual_apts = np.random.normal(choice1,pop_apt_std,num_individuals)
        idea_difs = np.random.normal(choice2,idea_dif_std,num_ideas)

        individual_knowledges_acquired = []
        for apt in individual_apts:
            expected_knowledge_acquired = 0
            for dif in idea_difs:
                expected_knowledge_acquired += knowledge_acquired(dif)*prob_acquire(apt,dif)
            individual_knowledges_acquired.append(expected_knowledge_acquired)    
    
        for i in range(len(individual_apts)):
            row = [individual_apts[i],individual_knowledges_acquired[i],choice1,choice2]
            plot_rows.append(row)
            

plot_df = pd.DataFrame(plot_rows,columns=['Individual Aptitude','Individual Knowledge','X','Y'])


#collapse-hide

selection = alt.selection_single(
    name='Select',
    fields=['X', 'Y'],
    init={'X': 0, 'Y': 0},
    bind={'X': alt.binding_radio(options=choices1), 'Y': alt.binding_radio(options=choices2)}
)


alt.Chart(plot_df).mark_circle().add_selection(selection).encode(
    x='Individual Aptitude:Q',
    y='Individual Knowledge:Q',
    tooltip='Title:N',
    opacity=alt.condition(selection,alt.value(1.00),alt.value(0.00))
)
